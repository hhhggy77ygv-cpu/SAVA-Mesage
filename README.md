# SAVA Messenger

SAVA Messenger — современный мессенджер с несколькими частями: сервер, веб-клиент и десктопное приложение.

## Структура проекта

- `apps/server` — backend, API и работа с базой данных
- `apps/web` — веб-клиент
- `apps/desktop` — Electron-версия для Windows
- `certs` — SSL-сертификаты
- `sounds` — звуковые файлы для приложения

## Быстрый запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите сервер и веб-клиент в режиме разработки:

```bash
npm run dev
```

3. Соберите проект для продакшена:

```bash
npm run build
```

4. Соберите десктопную версию:

```bash
npm run build:desktop
```
