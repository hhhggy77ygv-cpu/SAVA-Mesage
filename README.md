# SAVA Messenger

SAVA Messenger — современный кросс-платформенный мессенджер с серверной, веб и десктоп версиями.

## Структура

- `apps/server` — backend с настройками, базой данных и API
- `apps/web` — веб-клиент
- `apps/desktop` — десктопное приложение на Electron
- `certs` — SSL-сертификаты
- `sounds` — звуковые файлы для приложения

## Быстрый старт

1. Установить зависимости:

```bash
npm install
```

2. Запустить локально:

```bash
npm run dev
```

3. Собрать проект:

```bash
npm run build
```

4. Собрать десктопную версию:

```bash
npm run build:desktop
```

## Примечания

- `node_modules/` и сгенерированные файлы не должны попадать в репозиторий
- Файл `.gitignore` уже настроен для исключения зависимостей, билд-выхода и конфигураций окружения
- Если будете публиковать репозиторий на GitHub, задайте правильный URL репозитория в `package.json` при необходимости
